Todolist API
具備基礎 CURD 功能的代辦事項列表 API
  • id： 事項 id
  • content： todo 事項
  • state： todo 狀態，0 為未完成，1 為已完成

獲取所有todo

Example URI

GET http://fan630.com.tw/todo/api/api.php

<curl -X GET fan630.com.tw/todo/api/api.php>

/{:id}

獲取單一todo

Example URI

GET http://fan630.com.tw/todo/api/api.php/{id}

<curl -X GET fan630.com.tw/todo/api/api.php/91>

/{:id}

刪除單筆todo

Example URI

GET http://fan630.com.tw/todo/api/api.php/{id}

<curl -X DELETE fan630.com.tw/todo/api/api.php/92>

新增todo

Example URI

POST http://fan630.com.tw/todo/api/api.php/

Request with Body

POST http://fan630.com.tw/todo/api/api.php/

<curl -X POST fan630.com.tw/todo/api/api.php -d "content=homework">

/{:id}

修改單筆todo

Example URI

POST http://fan630.com.tw/todo/api/api.php/{id}

Request with Body

<curl -X PATCH fan630.com.tw/todo/api/api.php/93 -d "revise=修改成功&state=1">
