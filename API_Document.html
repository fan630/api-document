Todolist API
具備基礎 CURD 功能的代辦事項列表 API
  • id： 事項 id
  • todo： todo 事項
  • state： todo 狀態，0 為未完成，1 為已完成

獲取所有todo

Example URI

http://fan630.com.tw/todo/api/api.php

curl -X GET fan630.com.tw/todo/api/api.php
/{:id}

獲取單一todo

Example URI

http://fan630.com.tw/todo/api/api.php/139

curl -X GET fan630.com.tw/todo/api/api.php/139
Response 200 
                                          
    [
      {
        "id": 139,
        "todo": "homework",
        "state": 0,
      }
    ]
/{:id}

刪除單筆todo

Example URI

http://fan630.com.tw/todo/api/api.php/139

curl -X DELETE fan630.com.tw/todo/api/api.php/139

新增todo

Example URI

http://fan630.com.tw/todo/api/api.php

Request with Body curl -X POST fan630.com.tw/todo/api/api.php -d "content=homework"
Response 200 
                                          
    [
      {
        "result":"success"
        "id": 140,
        "content": "homework",
      }
    ]
/{:id}

修改單筆todo

Example URI

http://fan630.com.tw/todo/api/api.php/

Request with Body curl -X PATCH fan630.com.tw/todo/api/api.php/140 -d "revise=修改成功&state=1"
Response 200 
                                          
    {
        "result":"success",
        "message":"更新成功"
    }
